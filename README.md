# dotfiles

Laptop dotfiles

- Catppuccin-mocha colorscheme
- dwm bindings
- GTK3, QT5 and QT6 (qt5ct, qt6ct, kvantum)
- Alacritty
- Sway
- Waybar
- Swaync
- dmenu
- wofi
